# 2DS-AMS
 Portfólio de Atividades - 2º Ano

Este repositório reúne os principais trabalhos desenvolvidos durante o meu segundo ano do ensino médio/técnico.  
Ele contém projetos de diferentes áreas da programação, incluindo desenvolvimento de aplicações, lógica e modelagem de sistemas.

---

## Conteúdos do Repositório

- Diagramas de Caso de Uso (UML)
- Diagramas de Club Social
- Atividades em Java
- Aplicações desenvolvidas em Kotlin
- Projetos em PHP
- Aplicativo de cadastro utilizando SQLite

---

## Tecnologias Utilizadas

Durante o desenvolvimento dos projetos, foram utilizadas as seguintes tecnologias:

- ☕ Java
- 🟣 Kotlin
- 🐘 PHP
- 🗄️ SQLite
- 📊 UML (Modelagem de sistemas)

---

## Objetivo do Repositório

Este repositório tem como objetivo documentar minha evolução acadêmica e prática em programação,  
mostrando o aprendizado em diferentes linguagens, conceitos de desenvolvimento de software e boas práticas de organização de código.

---

## Organização

Os projetos estão separados em pastas de acordo com a linguagem ou tipo de atividade, facilitando a navegação e o entendimento do conteúdo.

---

## Sobre o aprendizado

Ao longo deste ano, foram explorados conceitos como:

- Programação orientada a objetos
- Estruturação de sistemas
- Desenvolvimento de aplicativos móveis
- Persistência de dados com banco SQLite
- Modelagem de sistemas com UML
